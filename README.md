## Build and run
```shell
git clone https://github.com/abdalrahmanshaban0/Islamic-Prayer-Timings/

cd Islamic-Prayer-Timings
mkdir build
cd build
cmake ..
make

# non-daemon
./IPT

# daemon
./IPT -d
  #OR
./IPT --daemon
```
